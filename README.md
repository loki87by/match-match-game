# **Match match game** 
Игра на запоминание написанная на TypeScript + WebPack 

В проекте используются следующие технологии:

* Семантические теги
* Адаптивная вёрстка
* Flex- и grid-контейнеры
* IndexedDB
* Относительные величины
* и многое другое
